----------------------------------
- This program converts the OS based program into a menu-driven program using Python Code which will execute the required user query when the user will give the input as a text.

libraries required:
---------------------------------

- If you recieve errors such as No module named win32com.client, No module named win32, or No module named win32api, you 
  will need to additionally install pypiwin32
